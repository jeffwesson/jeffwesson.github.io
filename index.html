Hello, I'm Jeff

I have been building websites for nearly three decades using a variety of development tools; from static HTML & CSS sites, to Content Management Systems using the LAMP stack, and Single Page Applications using the MEAN and MERN stacks.

I currently work within the React ecosystem creating rich Web Applications using Next.js, Tailwind, GraphQL, and RabbitMQ.

The Web has proven to be a very powerful tool. And now, with the the number of mobile devices running modern browsers and an Internet of Things becoming more commonplace, the Web and related technologies look poised to gain an even larger market share of data.

I hold a bachelors degree of anthropology which can help me tease out valuable information on how people use the products and tools we create. Putting this information to use throughout the entire development process enables me to better understand what users expect and why.

You can find periodic posts on my blog or follow me on the social media platforms listed below.
